Installation

To install BSR, run the setup program, setup.exe. Follow through the dialog boxes, and at the end click "Close".











Desktop Changes

The setup program adds an icon to the desktop and start menu.

Installed Files

The default installation directory is:
  • C:\Program Files\Mossywell\BSR
As of version 1.7.0, the files installed into this directory are:

File Details
README.TXT Last minute and version information about the application not present in the help pages.
Mossywell.BSR.exe The application itself.
Mossywell.WebUtils2.dll A supporting DLL that handles the HTTP communications asynchronously and which is responsible for marshalling calls back onto the UI thread.
Mossywell.NetUtils1.dll A supporting DLL that obtains detailed routing table information (more detail than is provided by the framework).
Mossywell.BSR.exe.config The XML configuration file. This application uses XML rather than the registry to store information as this is the standard being pushed by Microsoft for .Net applications.
BSR.chm This help file!
Bounce.wav A wav file that is played (if it exists) when the router is rebooted automatically. Replace this with your favourite router reboot sound. (You do have one don't you?)

Registry Settings

The Windows Installer saves a plethora of registry settings. They're not listed here, but a trawl for the word "mossywell" in the registry will dig them out.

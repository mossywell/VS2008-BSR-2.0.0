Uninstalling BSR

To uninstall BSR, it is best (by far) to use "Add or Remove Programs" in the Control Panel (or Programs and Features in Vista). BSR appears as "BSR" in the list of applications.

Uninstallation removes all relevant registry entries and files with one exception: the log files are not removed. If you then install BSR, the current log files are kept and used in the new installation (if the date is the same). The default location of the log files on Windows XP is "C:\Documents and Settings\<user name>\My Documents\Mossywell\BSR".

In addition, if uninstallation is attempted whilst BSR is running, Windows Installer will close the application first.

If you run into problems uninstalling, use the information in the Installation Details section to strip out any settings manually.
